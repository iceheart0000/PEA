# Zbiór algorytmów w C++ do problemu ATSP zaprojektowanych w ramach przedmiotu - Projektowanie Efektywnych Algorytmów.

## Zaprojektowane algorytmy w ramach danych zadań:
1. Brute Force, Nearest Neighbor, Repetitive Nearest Neighbor, Random Search
2. Branch & Bound z metodami przeszukiwania przestrzeni stanów: Best First-Search; Breadth First-Search.
3. Symulowane Wyżarzanie (SA - Simulated Annealing)
4. Algorytm Genetyczny (GA - Genetic Algorithm)
